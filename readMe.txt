repo ini merupakan repo untuk saya belajar git dan github.
rencananya sih mau diisi command-command git dan pengertiannya

jika anda melakukan perubahan pada file dan ingin memasukkan ke github maka ini caranya
pertama di add dulu perubahannya

git add .

lalu setelah itu di commit dengan menambahkan message untuk memperjelas maksud commitnya apa
git commit -m "ini untuk message"

setelah itu baru push sesuai branch
git push -u origin master

**NOTES**
agar tidak nabrak-nabrak, lebih baik pada setiap commit, diberi branch baru, lalu di merge saja.
